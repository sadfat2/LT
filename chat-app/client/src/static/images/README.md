# 图片资源说明

请在此目录放置以下图片资源：

1. `default-avatar.png` - 默认头像（建议尺寸 200x200）
2. `tab-message.png` - 消息 Tab 图标（建议尺寸 48x48）
3. `tab-message-active.png` - 消息 Tab 激活图标
4. `tab-contacts.png` - 通讯录 Tab 图标
5. `tab-contacts-active.png` - 通讯录 Tab 激活图标
6. `tab-profile.png` - 个人中心 Tab 图标
7. `tab-profile-active.png` - 个人中心 Tab 激活图标

图标颜色建议：
- 普通状态：#999999
- 激活状态：#07C160（微信绿）
